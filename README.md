# T20 Chasing Advantage Analysis

An empirical analysis of whether chasing provides an advantage in T20 cricket using IPL match data from 2008–2024.

## Project Overview

This project investigates whether teams batting second (chasing) are more likely to win T20 cricket matches.

The analysis includes:
- Comparison of win percentages between batting first and chasing teams
- Toss decision analysis
- Logistic regression modelling
- Seasonal trends in chasing success
- Venue-based chasing advantage analysis

## Files Included

- `index.html` – exported HTML version of the project website/blog
- `blog.ipynb` – Jupyter Notebook containing the full analysis and visualisations
- `matches.csv` – IPL dataset used for the analysis
- `README.md` – project overview and instructions

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- Jupyter Notebook

## Running the Project

Open `blog.ipynb` in JupyterLab or Jupyter Notebook and run all cells from top to bottom.

Alternatively, open `index.html` in a web browser to view the completed blog-style project output.

## Main Finding

The analysis suggests that chasing teams win slightly more often than teams batting first, indicating a moderate chasing advantage in T20 cricket.
